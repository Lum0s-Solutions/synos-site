## v111.0.0 — Last Light

**v111.0.0 "Last Light"** completes the v101→v111 **"Rust Everything"** campaign — a Linux 7.0 Rust-for-Linux kernel port with **33 signed Rust kernel modules** (ZeroC: zero Syn_OS-authored C). ALFRED runs at **v6.0**, the GRIMOIRE catalog holds **117 labs** across 13 categories, the Rust workspace stands at **239 crates** with 0 compile errors, and post-quantum cryptography (ML-KEM, ML-DSA, SLH-DSA) is **on by default** across every transport, signing path, and audit trail. GRIMOIRE is now a **playable shared-world hacking MMO** where the exploit *is* the objective — [play it in your browser →](/play/).
