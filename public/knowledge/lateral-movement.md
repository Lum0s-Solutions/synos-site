# Lateral Movement

Lateral movement is the set of techniques an attacker uses to progressively move through a network after gaining an initial foothold, pivoting from host to host to reach high-value targets. It commonly leverages stolen credentials, pass-the-hash, remote services such as RDP, SMB, WMI, and PsExec, and Active Directory abuse. It corresponds to the TA0008 tactic in the MITRE ATT&CK framework.
